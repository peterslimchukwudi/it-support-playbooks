# AnyDesk Remote Support Procedure

## Issue
User unable to resolve software issue without remote assistance.

## Tools Used
- AnyDesk
- Windows 10/11

## Steps Taken
1. Verified user identity before session.
2. Requested AnyDesk ID and permission.
3. Connected securely to user system.
4. Diagnosed issue and applied fix.
5. Tested resolution with user.

## Result
Issue resolved remotely, minimizing downtime.

## Best Practices
- Always confirm user consent
- End session immediately after resolution
